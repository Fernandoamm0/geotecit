Ingreso de Equipo a Laboratorio

Sistema de control de entrada de equipo a laboratorio
Desarrollado por Ing. Fernando Mora

📘 Manual de Uso Operativo

1. Orden de Reparación y Fecha:
El folio se genera automáticamente (GT + Año + Sem + Consecutivo). El sistema reinicia el contador a 001 al inicio de cada semana para mantener el orden.

2. Identificación del Equipo:
Seleccione la Marca. Si el Número de Serie o Parte es ilegible, escríbalo como "ILEGIBLE".

3. Falla Reportada:
Transcriba el reporte exacto de la guía o documento de envío. Si no hay reporte, indique "Sin reporte de falla".

4. Procedencia e IDC:
Indique de qué estado proviene el equipo y el nombre o identificador del Técnico de Campo (IDC) que lo envía. (Use "Almacén interno" para rescates).

5. Criterios de Estatus Inicial:

  • Reparable: Daño estándar, apto para proceso.
  • Irreparable: Daño catastrófico visible (quemado, fractura severa).
  • Intervenido: Evidencia de manipulación no autorizada o sellos rotos.
  • Piezas faltantes: El módulo llega incompleto (faltan tornillos, tarjetas, etc).
